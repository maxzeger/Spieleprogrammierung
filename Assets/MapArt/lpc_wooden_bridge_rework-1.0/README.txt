Name:
  LPC Wooden Bridge Rework
Version:
  1.0
Description:
  LPC wooden bridge tileset by Xenodora reworked as drop-in replacement
  for Stendhal.
Tileset details:
  - Created for use with 32x32 tiled maps.
  - Orientation: orthogonal
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
